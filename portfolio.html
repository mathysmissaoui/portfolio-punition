groupie tracker

Création d'une application web en Go qui se connecte à une API de musique. Les utilisateurs peuvent rechercher des chansons, des artistes ou des albums. Les fonctionnalités peuvent inclure la création de listes de lecture, le partage de musique et la suggestion de contenu basée sur les préférences de l'utilisateur.
